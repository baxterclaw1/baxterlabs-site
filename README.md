# BaxterLabs Website

Redesign of [baxterlabs.io](https://baxterlabs.io). Astro static site.

## Control docs

| File | Purpose |
|------|---------|
| `PROJECT.md` | Rules, workflow, skill order |
| `BRIEF.md` | Positioning and vibe (fill in session 1) |
| `DECISIONS.md` | Locked choices only |
| `CONTENT.md` | Draft copy warehouse |

## Commands

```bash
npm install
npm run dev      # http://localhost:4321
npm run build    # output to dist/
npm run preview  # preview production build
```

## Workflow

One section per session. Read `PROJECT.md` before building. Hero first.

## Deploy (later)

Connect repo to Vercel for staging preview. Do not change baxterlabs.io DNS until staging is approved.
